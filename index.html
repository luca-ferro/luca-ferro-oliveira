Proposta do Website

Este website é o terceiro e último trabalho da disciplina de Programação Web, lecionada pelo professor Rodrigo Martins Pagliares. Nele, devíamos criar nosso site pessoal baseado no website criado em Wordpress. Após isso, devíamos hospedar o site em algum servidor de hospedagem gratuita ou no servidor Turing. Essa atividade tem o intuito de ensinar aos alunos como utilizar das linguagens HTML e CSS, tal como demonstrar quais as facilidades e dificuldades que enfrenta um desenvolvedor Front-End.
